# eV
Minimal viewer for EGSnrc simulations
-------------------

Use eV to view:
- Simulation geometries
- Particle sources
- Dose distributions
- Particle paths  

-------------------

Planned input files:
- `.egsinp`
- `.egsdat`
- `.ptracks`

--------------------
