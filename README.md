# eV
Minimal viewer for EGSnrc simulations
-------------------

Use eV to view:
- Simulation geometries
- Particle sources
- Dose distributions
- Particle paths  
-------------------

Accepted file inputs:
- `.egsinp`
- `.egsdat`
- `.ptracks`

--------------------
